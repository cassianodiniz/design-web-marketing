# Changelog — design-web-marketing

## 0.1.0 — 2026-06-17

- Criação do plugin `design-web-marketing`.
- Skill `maestro`: orquestrador de landing/site/marketing de conversão, com roteamento,
  pipeline completo (contexto → oferta → copy → construir → revisar → publicar) e travas
  que adaptam voz e modelo de negócio ao contexto do usuário (não ao SaaS genérico das
  sub-skills). Avaliada no harness do skill-creator: ~15/15 de roteamento correto vs 6/15 sem a skill.
- Empacotadas as 7 skills de marketing adaptadas de coreyhaines31/marketingskills (MIT):
  product-marketing, offers, copywriting, copy-editing, cro, marketing-psychology,
  customer-research. Crédito em CREDITS.md + LICENSE-marketingskills.
